进阶 git 多仓库管理开发（二）

Feb 12, 2017

接着上次的多仓库管理开发(一)，在很多情况下我们会涉及到不同的仓库使用不同的用户名和email，比如：在公司开发的时候一般都是使用自己真实名字和公司邮箱；在github时候一般不会暴露自己真实的姓名。这个时候多仓库设置不同用户名就非常有必要了。

在多说一遍，在每个git仓库下，都有一个.git/ 文件夹，使用vim或者其他工具，打开.git/config 文件，添加一下内容：

[user]
    name = 用户名
    email = 邮箱地址

这样就解决问题了，当然你也可以使用命名行的形式指定用户名：

加上 —global 表示设置全局的，对所有的项目都有效，去掉 —global 表示针对某个特定的项目。
