This is a demo.

这是一个测试Demo，并不是使用繁琐的document.write或者innerHTML来实现。这些古老的方式，要么在HTML文档中插入<script> 标签，要么无法对插入的内容进行处理。而DOM提供了更加强大更加精准的功能，在任何时候都能替代innerHTML。

index.html
<!DOCTYPE html>
<html>
<head>
   <meta charset="utf-8" />
   <title></title>
</head>
<body>
   <div id="txtdiv"></div>
   <script type="text/javascript" src="js/myFunction.js"></script>
</body>
</html>

平稳退化

<li>
    <a href="images/wallpaper_01.jpg" title="wallpaper_01" onclick = "showPic(this); return false;">one</a>
</li>

伪代码

<li>
    <a href="javascript:showPic("images/wallpaper_01.jpg"); return false;" title="wallpaper_01" >one</a>
</li>

如果浏览器禁用javascript，伪代码就会无法显示。

体验之行

第一次体验店体验

第一次去体验店体验还在上高中的时候，依稀记得，那时候国产手机刚刚崛起，国内还是三星称霸的时候。魅族凭借着MX2和MX3，足足火了一把，记得那时和小米竞争地非常激烈（还有红衣和雷布斯的撕逼，蹦出个360特供机）。嘉善迎来了第一家魅族体验店，朋友带着我去的，当时也是抱着去玩玩的心情，体验一下MX3，尽管网上对它的评价褒贬不一。特别害羞地进去摸了下真机，然后工作人员给我吹了各种逼。当时第一感觉，非常小清新，做工还行，体验还算不错的。后来我就给我朋友推荐了MX3，之后朋友告诉我她的使用感受的时候，我觉得我错了。当时网上也爆出魅族的售后、做工一些列问题，终究还是小企业。毕竟没有三星、曾经Nokia那样的产业链和售后服务。

总结: 在你不了解一个品牌它的尿性的情况下，你不要盲目的去推崇它。毕竟姜还是老的辣。
