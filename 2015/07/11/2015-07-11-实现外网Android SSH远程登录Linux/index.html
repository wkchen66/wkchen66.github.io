实现外网Android SSH远程登录Linux

Jul 11, 2015

这个想法已经有一整子了，一直内沉下心来去做，今天台风正好闲着没事做。其实原理很简单，通过路由器（TP-Link WR740N）转发设置DMZ主机，让局域网中的一台计算机完全暴露给广域网，接着用动态DNS解析到主机。在手机端使用SSH登入Linux，这里推荐使用JuiceSSH 。
以下介绍两款linux安装方法:

1.安装ssh server

ubuntu

sudo apt-get install openssh-server

ArchLinux

sudo pacman -S openssh

2.查看ssh服务是否启动（一般情况下，安装完毕后自动启动）

ps -ef | grep sshd

3.如果发现无sshd这个进程，则需要手动启动

ubuntu

sudo /etc/init.d/ssh start

ArchLinux

sudo /etc/rc.d/sshd start

4.仅供测试配置文件先不做修改，默认端口22

Tip:提醒一下修改了配置文件，需要重启服务restart一下

5.设置路由器

开启DMZ状态，设置DMZ主机IP为从DHCP获得的主机IP地址。
注册花生壳帐号，勾选“启用DDNS”登入。这里记住你注册的域名。

6.测试登入

在手机上测试ping 域名，看能否ping通。
尝试登入，ssh:域名 port:22 user:linux用户 passwd:linux用户密码
登入成功。
