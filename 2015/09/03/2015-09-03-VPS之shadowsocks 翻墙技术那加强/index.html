VPS之shadowsocks 翻墙技术那加强

Sep 3, 2015

目录

  • shadowsocks
  • openWrt for Hiwifi

shadowsocks

  • 1 购买vps

这里推荐有日本节点的vps延迟比较低，例如Linode 、Conoha(价格相对比较贵)

  • 2 安装shadowsocks 以及相关依赖（python版）

    wget –no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh
    chmod +x shadowsocks.sh
    ./shadowsocks.sh 2>&1 | tee shadowsocks.log

  • 3 配置shadowsocks服务

修改vim /etc/shadowsocks.json 文件

如下:

{
“server”:”0.0.0.0”,
“server_port”:8989,
“local_address”:”127.0.0.1”,
“local_port”:1080,
“password”:”password”,
“timeout”:300,
“method”:”aes-256-cfb”,
“fast_open”: false
}
多用户如下：

{
“server”:”0.0.0.0”,
“local_address”:”127.0.0.1”,
“local_port”:1080,
“port_password”:{
“8989”:”password0”,
“9001”:”password1”,
“9002”:”password2”,
“9003”:”password3”,
“9004”:”password4”
},
“timeout”:300,
“method”:”aes-256-cfb”,
“fast_open”: false
}

  • 4 启用服务

    /etc/init.d/shadowsocks start //启动
    /etc/init.d/shadowsocks stop //停止
