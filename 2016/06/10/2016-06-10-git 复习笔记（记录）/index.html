git 复习笔记（记录）

Veröffentlicht am 2016-06-10   |  

认识git已经一年多了，但一直没有正真的弄懂它。第一接触到时当时想要搭建博客，很幸运遇到了hexo（基于node.js的静态博客），其中涉及到了一些基本的命令，虽然在android和java开发的时候偶尔会去提交一些代码到github上（其实就三句git add、git commit、git push其他的都忘了），但是说真的压根不理解这玩意。直到最近github对我越来越重要，于此同时又是hexo,博客更新停止了有一段时间了，最近又打算开启每天写博客的习惯了，自学android差不多快一年了，但是进展让我不够满意，所以我打算每天写博客记录我的学习过程，也提醒自己坚持下去。本文配套github做的记录。

git init //初始化仓库

git status //查看git状态

git add //添加文件到本地库

git add .  //添加该文件夹下的所有文件

git commit -m //提交信息

git log //查看git log日志

git branch 分支默认master

git branch   //查看当前的所有分支
git branch 分支名字   //创建该分支
git checkout 分支名字   //切换到该分支
git checkout -b 分支名字   //创建并切换到该分支

git pull origin master (分支) //从远程库拉取同步到本地

git push origin master (分支) //从本地同步到远程库

git clone git@github.com:username/仓库名.git //克隆改仓库到本地
