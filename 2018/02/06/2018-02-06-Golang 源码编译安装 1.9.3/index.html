Golang 源码编译安装 1.9.3

Feb 6, 2018

今天在线上服务器搭建 Go 环境的时候，遇到了一些问题，编译 go1.9.3 的时候出现了

yum -y install git

git clone https://github.com/golang/go.git

cd /usr/local/go/src

./all.bash

cd /usr/local/go1.4
git checkout go1.4

cd /usr/local/go1.4/src
./all.bash

cd /usr/local/go/src
./all.bash

golang

export GOPATH=/usr/local/go/bin

source ~/.bashrc

  1. /etc/profile系统级别，对系统下的所有用户有效
  2. .bash_profile 、.bashrc和.zshrc仅对当前登录用户有效
